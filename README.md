### Run API

```bash
# Go to server folder
$ cd Proffy/server

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Access API at http://localhost:3333/

### Run Web Project

```bash
# Go to web folder
$ cd Proffy/web

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
